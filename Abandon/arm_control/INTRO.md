# 总线舵机配置

`S1 底座旋转（腰关节，ID=1）`

位置：机械臂底座与小车云台之间

运动范围：0°～180°（水平旋转）

功能：整体水平旋转，扩大作业方位
扭矩：15kg·cm（支撑全臂）

`S2 肩部（ID=2）`

位置：旋转台上方，连接大臂

运动范围：0°～180°（俯仰）

功能：控制大臂升降，决定臂高与作业半径

扭矩：15kg·cm（承受大臂 + 小臂 + 负载）

`S3 肘部（ID=3）`

位置：大臂与小臂之间

运动范围：0°～180°（屈伸）

功能：小臂折叠 / 伸展，调节末端远近

扭矩：15kg·cm（承受小臂 + 负载）

`S4 腕部俯仰（ID=4）`

位置：小臂末端，腕关节第一段

运动范围：0°～180°（俯仰）

功能：调整末端上下姿态

扭矩：15kg·cm

`S5 腕部旋转（ID=5）`

位置：S4 与夹爪之间，腕关节第二段

运动范围：0°～180°（旋转）

功能：末端无限位旋转，调整夹爪朝向

扭矩：15kg·cm

`S6 夹爪（ID=6）`

位置：最末端，夹爪驱动

运动范围：30°～180°（开合）

功能：夹持 / 释放物体，最大开口约 80mm

扭矩：15kg·cm（夹紧力可调）
